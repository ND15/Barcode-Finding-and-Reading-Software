# Barcode-Finding-and-Reading-Software

## Overview
This repository contains the implementation of barcode finding and reading software.

## Requirements

### Supported Versions
- Python 3.3+

### Libraries
- pyzbar
- openCV
- imutils
- numpy

## Run Code
Command::
  python filename.py --dataset path_to_the_dataset
